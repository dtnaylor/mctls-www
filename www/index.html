multi-context TLS (mcTLS)

mcTLS is a secure communication protocol that extends TLS to allow endpoints to incorporate trusted middleboxes into secure sessions.

  • No transparent proxies—explicit control for both endpoints
  • Access control for middleboxes—they see only what they need to do their jobs
  • All parties can be authenticated
  • No custom root certificates
